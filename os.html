Ajuda OS

Assim que a tela inicial da aplicação estiver aberta para ter uma visão geral das ordens basta visualizar a parte superior do sistema conforme a imagem abaixo:

A parte superior apresenta as informações sobre as OS (tanto incidentes quanto requisições) e seus status atualmente.

Para visualizar informações mais detalhadas sobre as OS basta selecionar a função "Editar" ou acessar a tela "Relatórios".

A tela de OS oferecem dois botões o de "Criar" uma nova OS ou "Editar" uma existente.

Ao selecionar a opção "Criar" a seguinte tela irá aparecer.

Ela serve para validar a já existência de um cliente no sistema não sendo necessário assim um novo registro. Caso o registro já esteja criado a tela vai atualizar perguntando se o usuário deseja efetuar uma nova consulta ou utilizar o cadastro localizado.

Caso não esteja Cadastrado uma nova tela irá aparecer para registrar o requisitante da OS

Para mais informações de cadastro de requisitantes consultar a ajuda via ChatBot na opção de "Cadastro e edição de cliente"

Ao utilizar o cadastro existente uma tela onde deve ser inseridas as informações da OS aparecerá.

No opção Situação pode escolher entre usar um equipamento já existente no sistema ou criar um novo. Ao selecionar Cadastrar será solicitado a inserção dos campos Fabricante do equipamento, Modelo e Preço em R$.

No opção Fornecedor deve colocar se o equipamento é de posse da própria Eleva ou de terceiros.

Na seção Dados da ordem de serviço deve designar o funcionários Responsável pela OS.

Definir se a OS será do tipo Requisição ou Incidente.

Pode colocar uma breve Descrição com texto livre.

Por fim deve definir qual Status da OS se vai ficar em aberto, se já irá ser executada de imediato ou se é apenas um registro de uma OS já finalizada.
